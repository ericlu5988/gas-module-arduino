# gas-module-arduino

## Python Gas Module with Arduino

### Installation

1. Clone this repository and open the .ino file using Arduino IDE
2. Ensure dependencies are met
3. Compile and Run

### Dependencies

#### LiquidCrystal_I2C.h
Install via Arduino IDE > Tool > Manage Libraries > Seach for Liquid Crystal
